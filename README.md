## 這是什麼？

市面上多數 財務自由/退休試算工具，假設報酬率永遠固定（例如「年化8%」），算出一條漂亮的成長曲線。但真實市場不會逐年穩定上漲——退休後第一年遇到空頭跟退休後第二十年遇到空頭，對資產存活率的影響天差地遠，這就是財務學界所稱的**報酬順序風險（Sequence of Returns Risk）**。

「**財務自由永動機試算**」用一個單一HTML檔案，同時提供三種視角：

1. **固定報酬率模式**：傳統試算邏輯，快速、直覺，適合日常微調參數、核對公式細節。
2. **PMC隨機市場模擬**：用參數化蒙地卡羅方法，重複模擬數千次不同的市場路徑，告訴你「在真實世界的波動下，這套提領計畫撐過60年的機率有多高」。
3. **歷史回測**：不用統計分布生成隨機數字，直接套用1970~2025年S&P 500真實發生過的逐年報酬率序列，讓你看到「如果正好在2008年金融危機那年退休」這類真實情境會發生什麼事。

無論你已經退休、即將退休，還是正在累積資產、計畫提早財務自由（FIRE），都可以用這個工具測試自己的計畫禁不禁得起市場的隨機考驗。

## 核心特色

### 📊 兩套資產情境，貼合台灣常見退休方式
- **純現金模式**：賣房、繼承、儲蓄、退休金一次提領——任何一次性現金來源都適用，也可疊加「已持有、無貸款的房產」。
- **房貸／信貸模式**：房子不賣、增貸投入市場，同時模擬貸款攤還（含寬限期、升息壓力測試）。

### 🎯 兩種目標哲學
- **永動機模式**：本金不歸零，理想上可以一直提領下去。
- **歸零模式**：花到用盡也沒關係，搭配「通膨樓地板」機制確保現金流不會輸給物價上漲。

### 💳 三種提領邏輯
固定比率、固定金額、通膨調整提領（依市場真實表現決定要不要加薪/減薪），各自有清楚的數學公式跟適用情境說明。

### 🎲 PMC蒙地卡羅壓力測試
- 對數常態分布隨機抽樣，1,000~10,000次模擬任選。
- 雙指標呈現：**資產存活率**（錢有沒有花完）與**購買力達標率**（花的時候夠不夠用）——兩者經常有顯著差距，正是固定報酬率模式完全看不到的真實風險。
- 提領調整機制改用「資產覆蓋率」判斷（參考自退休理財領域知名的Guyton-Klinger護欄法則精神），避免市場單年劇烈震盪造成提領金額無意義地暴漲暴跌。
- P10~P90資產分布扇形圖，一眼看出「最壞10%」到「最好10%」情境的資產路徑範圍。

### 📜 歷史回測（真實數據，不靠假設）
- 直接使用1970~2025年S&P 500指數真實年度總報酬率（含股息），每次模擬隨機抽取連續N年的真實歷史片段套用到你的計畫上。
- 「最差案例」會明確標示對應的真實起始年份（例如「西元1997年起」），比抽象的統計模擬更有說服力。
- 誠實標註限制：僅用美股歷史數據，債券部分仍為簡化假設，過去不代表未來。

### 🧮 資產配置細分（股債比例取代抽象波動度）
- 不需要自己猜測「投資組合波動度」這種抽象數字，只要設定「股票佔多少比例」，系統自動用變異數加權公式算出混合後的預期報酬率與波動度。
- 股票比例設100%時債券完全不影響結果，反之亦然——配置邏輯完全透明，點一下就能看到完整計算過程。

### 📸 情境快照比較
- 把目前這組參數設定連同檢核結果存成一張快照卡，最多同時保留3組，並排顯示。
- 適合比較「65歲退休」vs「60歲退休」、不同提領率等情境差異，平時不佔版面，需要時才展開。

### 🛡️ 誠實的審核機制
不會自己騙自己「達標」——「可持續性檢核」用三項硬標準（現金流是否跟上通膨、本金是否歸零、期末資產是否健康）給出通過/部分通過/未通過的明確結論，並清楚區分「估算參考值」與「真正耗盡」兩種不同概念，避免使用者誤判。

### 📱 真正貼心的使用體驗
- 字體大小可調（100%~500%），照顧長輩與視力不便的使用者。
- 所有參數自動儲存在你自己的瀏覽器（不上傳任何伺服器），重新整理也不會被清空。
- 一鍵列印成PDF報表（含目前選擇的模式與滑桿設定，圖表完整呈現），逐年明細表格可左右滑動查看完整欄位，也可一鍵複製成Excel/Google Sheets可直接貼上的格式。
- 超過60項參數都配有白話說明與計算範例，點一下ⓘ就能看懂每個數字怎麼來的。

## 怎麼用？

1. 下載 `retirement_planner.html`（或本 repo 內對應檔案）。
2. 用瀏覽器直接開啟（雙擊檔案，或瀏覽器內「開啟檔案」）。
3. 依照自己的情況調整左側／上方的滑桿參數。
4. 查看「可持續性檢核」區塊的結論，或切換到「PMC隨機市場模擬」「歷史回測」做進一步壓力測試。
5. 滿意的話可以「儲存目前情境為快照」留存比較，或「產生PDF報表」「複製CSV」保存結果。
